# Simon Proa

Simon Proa es una version del juego Simon hecha con HTML, CSS y JavaScript.

El juego muestra una secuencia de colores. La persona que juega debe repetirla en el mismo orden. En cada acierto, la secuencia crece y el desafio aumenta.

Este proyecto se usa en clase como excusa para practicar trabajo colaborativo con Git, GitHub, VS Code, ramas y Pull Requests.

## Para observar

- La estructura de la pagina esta en `index.html`.
- El aspecto visual esta en `styles.css`.
- La logica del juego esta en `game.js`.
- Los sonidos estan en la carpeta `sounds/`.
