# Simon Proa

Simon Proa es una versión del juego Simon hecha con HTML, CSS y JavaScript.

El juego muestra una secuencia de colores. La persona que juega debe repetirla en el mismo orden. En cada acierto, la secuencia crece y el desafío aumenta.

Este proyecto se usa en clase como excusa para practicar trabajo colaborativo con Git, GitHub, VS Code, ramas y Pull Requests.

## Para observar

- La estructura de la página está en `index.html`.
- El aspecto visual está en `styles.css`.
- La lógica del juego está en `game.js`.
- Los sonidos están en la carpeta `sounds/`.
